# FDE-Full-Lifecycle — Checklist

## 准备

- [ ] 已加载 `references/fde-framework.md`
- [ ] 已确认客户授权与数据脱敏要求
- [ ] 干系人名单与访谈计划已排期
- [ ] 已运行或准备 `scripts/client_audit.py`

## Audit 执行

- [ ] 业务/技术/管理层访谈已完成
- [ ] As-Is 流程图已绘制
- [ ] 痛点矩阵（影响×可行性）已排序
- [ ] 数据资产与权限已盘点
- [ ] 技术环境约束已记录

## Evals 执行

- [ ] 业务需求已翻译为技术问题定义
- [ ] MVP/POC 范围已界定
- [ ] 核心路径 POC 已验证或计划明确
- [ ] ROI 与风险矩阵已输出
- [ ] 客户已对齐预期边界

## Deployment 执行

- [ ] 原型/系统集成方案可运行
- [ ] UAT 通过与验收记录齐全
- [ ] 灰度/上线与回滚路径已演练
- [ ] 运维手册与培训已完成
- [ ] 业务价值达成有量化或定性证据
- [ ] 私有化/AI Gateway/RBAC 上线门禁已通过（如适用）
- [ ] FDE-Adoption-Growth 采纳计划已交接

## FDE 阶段门禁

- [ ] PoC：Audit+Evals 输出场景卡与 PoC 计划
- [ ] Beta：Evals 评估集达标，周度 Demo 建立
- [ ] 生产：Deployment Runbook + RBAC-Audit + 采纳看板

## 复盘

- [ ] 产品化抽象检查三项已完成
- [ ] 可复用组件已登记
- [ ] 经验教训已写入项目复盘
