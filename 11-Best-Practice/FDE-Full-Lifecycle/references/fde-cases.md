# FDE 行业案例库

## 案例一：工业制造 — AI 视觉质检

### 背景
上海某消费电子工厂，产线换线调机耗时数小时，质检依赖人工目检，漏检率约 3%。

### FDE 工作过程

**审计阶段（2 周）：**
- 白天在产线采集异常样本（正常/缺陷各 5000 张）
- 理解产线流程：进料 → 贴片 → 回流焊 → AOI → 人工复检
- 发现关键问题：AOI 过杀率高（误报多），人工复检成为瓶颈

**评估阶段（1 周）：**
- 技术方案：基于 YOLOv8 的缺陷检测 + 分类模型
- POC 验证：500 张测试集，mAP@0.5 = 0.87
- ROI 计算：误报率降低 60%，预计节省 3 名质检员/产线

**部署阶段（3 个月循环迭代）：**
- 每周采集新样本，迭代模型
- 最终换线时间压缩至 55 秒
- 单日可换型号从 5 款扩展到近 300 款

### 关键启示
- 工业场景的核心瓶颈往往不是模型精度，而是**数据采集效率**
- 模型需要持续迭代，不是"部署完就走"
- 与产线工人建立信任关系至关重要（他们是最好的数据标注员）

## 案例二：智慧农业 — AI 精准喷洒

### 背景
John Deere 与 OpenAI 的 FDE 团队合作，目标降低农药使用量。

### FDE 工作过程

**审计阶段：**
- FDE 工程师驻场农田，理解农业喷洒完整流程
- 发现核心问题：传统喷洒是地毯式覆盖，实际只有 5% 面积有杂草
- 关键技术约束：拖拉机在农田移动，需要实时边缘推理

**评估阶段：**
- 技术方案：GPT-4V 助力杂草识别 + 实时决策系统
- 关键挑战：模型需要在边缘设备上运行（农田无稳定网络）
- POC：小规模农田测试，杂草识别准确率 92%

**部署阶段：**
- 化学制剂使用量降低 60%-70%
- 边缘推理延迟 < 200ms（满足实时喷洒需求）
- 模型持续学习，越用越准

### 关键启示
- 农业场景的核心是**边缘部署**，不是云端 API
- 需要理解物理世界约束（光照变化、天气、移动速度）
- ROI 计算要算"省钱 + 增产 + 环保"三维价值

## 案例三：金融 — AI 风控与合规

### 常见场景

```markdown
| 场景 | FDE 介入方式 |
|------|------------|
| 信贷审批 | RAG + 规则引擎，整合分散的政策文档 |
| 反洗钱 | NLP 提取交易备注中的风险信号 |
| 监管报告 | NLG 自动生成监管报告初稿 |
| 客服合规 | 实时质检，检测不当承诺/违规用语 |
```

### 典型工作流程

1. **审计**：理解监管要求（银保监会/央行的具体条文）
2. **评估**：确认数据可用性（客户信息、交易记录、通信记录）
3. **部署**：构建合规引擎（规则+NLP模型），嵌入业务流程

### 关键启示
- 金融 FDE 的第一优先级是**合规**，不是技术创新
- 可解释性比精度更重要（监管要求模型决策可追溯）
- 数据安全是硬约束（数据不出境、生产数据脱敏）

## 案例四：医疗 — AI 辅助诊断

### 典型挑战

- 数据高度敏感（HIPAA/个人信息保护法）
- 模型需要医疗器械注册
- 医生信任度是关键障碍
- 医院 IT 系统老旧，集成困难

### FDE 工作模式

```markdown
第一步：影子模式（Shadow Mode）
- 模型在后台运行，医生看不见
- 收集 3-6 个月数据，对比人机差异

第二步：辅助模式
- 模型给出建议，医生最终决策
- 收集医生反馈，持续优化

第三步：信任建立
- 准确率达到临床标准后，逐步推广
```

### 关键启示
- 医疗 FDE 需要**大量时间**建立信任
- 不是替代医生，而是为医生减负
- 监管合规是第一优先级

## 行业模式总结

### 各行业 FDE 特征对比

| 行业 | 典型痛点 | 核心能力 | 部署特点 | 周期 |
|------|---------|---------|---------|------|
| 制造 | 质检、排产 | CV、运筹优化 | 边缘部署 | 3-6月 |
| 农业 | 精准作业 | CV、IoT | 边缘+云端 | 6-12月 |
| 金融 | 风控、合规 | NLP、规则引擎 | 私有化部署 | 6-18月 |
| 医疗 | 辅助诊断 | CV、NLP | 医院内网 | 12-24月 |
| 零售 | 客服、推荐 | RAG、推荐系统 | 云端为主 | 1-3月 |
| 能源 | 预测维护 | 时序预测、IoT | 边缘+云端 | 6-12月 |

### 跨行业复用模式

以下能力具有跨行业通用性，产品化优先级最高：

1. **RAG 知识库**：知识问答场景（客服、内部培训、合规查询）
2. **智能质检**：视觉/文本异常检测（制造、金融、内容审核）
3. **自动化报告生成**：NLG + 模板引擎（金融、医疗、政府）
4. **智能排程**：多约束优化（制造、物流、能源）
5. **实时监控告警**：时序异常检测（能源、制造、运维）
