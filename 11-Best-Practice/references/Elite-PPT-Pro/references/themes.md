## 风格A：咨询青绿色（麦肯锡/BCG风格）

```javascript
const themeA = {
  primary:   "1A1A2E",  // 深藏青
  secondary: "2D6A6A",  // 深青绿
  accent:    "00A69C",  // 亮青绿（主强调色）
  light:     "E8F7F6",  // 浅青绿
  bg:        "FFFFFF",
  darkText:  "333333",
  midText:   "666666",
  border:    "DDDDDD",
};
```
