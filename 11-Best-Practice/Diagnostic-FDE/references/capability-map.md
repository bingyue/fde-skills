## 一、应用开发平台（ADP）

### 1.1 微服务与容器

| 产品 | 能力描述 | 适用场景 | 成熟度 |
|------|---------|---------|:-----:|
| **TSF（微服务框架）** | Spring Cloud/Service Mesh微服务治理，服务注册发现、配置管理、熔断限流 | 传统应用微服务改造、企业级服务治理 | ★★★★★ |
| **TKE（容器服务）** | K8s集群管理、弹性伸缩、CI/CD集成 | 容器化部署、弹性计算、DevOps | ★★★★★ |
| **Serverless** | 函数计算、事件驱动、按量付费 | 轻量级API、事件处理、定时任务 | ★★★★☆ |

## 二、WorkBuddy 能力集

### 2.1 核心能力

| 能力 | 描述 | 适用场景 | 成熟度 |
|------|------|---------|:-----:|
| **Skill系统** | 自定义工作流封装，SKILL.md + scripts/ + references/ 三层结构 | 方法论沉淀、重复性工作自动化 | ★★★★☆ |
| **Agent Memory** | 长期记忆存储（working memory files），跨会话上下文保持 | 持续性项目协作、知识积累 | ★★★☆☆ |
| **Ontology** | JSONL知识图谱，Entity/Relation CRUD + YAML Schema验证 | 知识建模、实体关系管理 | ★★★☆☆ |
| **MCP协议** | 连接外部SaaS系统的标准协议（Model Context Protocol） | 系统集成、数据互通、工具扩展 | ★★★★☆ |
| **插件系统** | 浏览器自动化、文件处理（PDF/XLSX/PPTX/DOCX）、Web Fetch | 数据采集、文档处理、自动化操作 | ★★★★★ |
| **Automation** | 定时/周期性任务自动执行（RRULE调度） | 定期报告、监控巡检、数据同步 | ★★★☆☆ |

### 2.2 Agent组件

| 能力 | 描述 | 适用场景 | 成熟度 |
|------|------|---------|:-----:|
| **Multi-Agent Team** | 多Agent协作（team_create + send_message），角色分工并行执行 | 复杂任务分解、多专家协作 | ★★★☆☆ |
| **Tool Calling** | AI调用外部工具（execute_command, web_search, web_fetch等） | 信息检索、代码执行、系统操作 | ★★★★★ |
| **Code Generation** | 代码生成与修改（read_file, write_to_file, replace_in_file） | 软件开发、脚本编写、配置管理 | ★★★★★ |
| **Image Generation** | AI图片生成（image_gen工具） | 营销素材、概念图、可视化 | ★★★☆☆ |

## 六、能力-场景快速匹配表

> 当客户描述痛点时，快速定位推荐的产品组合。

| 客户痛点类别 | 典型表述 | 推荐能力组合 | 方案模式 |
|------------|---------|------------|---------|
| **数据孤岛** | "系统之间数据不通" | MCP连接器 + DataWay + 向量DB + RAG | 数据集成+智能问答 |
| **人工重复操作** | "每天大量重复性工作" | WorkBuddy Skill + Agent + Automation | 流程自动化 |
| **决策缺数据** | "拍脑袋做决策" | 混元API + 向量DB + BI + 知识引擎 | AI辅助决策 |
| **供应链预测** | "库存总是不准" | 混元API + TI平台(时序预测) + DataWay | 智能预测补货 |
| **客户服务** | "客服成本太高/响应慢" | 混元对话 + 知识引擎 + WorkBuddy | 智能客服 |
| **文档处理** | "每天处理大量文档" | 混元多模态 + WorkBuddy插件(PDF/XLSX) | 智能文档处理 |
| **营销获客** | "获客成本越来越高" | 混元API + 数据分析 + 自动化营销 | 智能营销 |
| **合规风控** | "合规检查耗时耗力" | 混元API + 知识引擎 + 规则引擎 | 智能合规审查 |
| **研发提效** | "开发效率需要提升" | WorkBuddy + ADP + TKE + CI/CD | 研发效能平台 |
| **数据资产化** | "数据有价值但没利用" | WeData + CDWPG + 向量DB + 知识引擎 | 数据中台+AI |
